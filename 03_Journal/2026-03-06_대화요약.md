## 2. AG Agent Manager vs CC Agent Teams

### CC Agent Teams (2026.02.05 출시, Opus 4.6 포함)
- CC 내 네이티브 기능. 독립 컨텍스트 창 × N개 병렬 실행
- Team Lead가 Teammate 직접 조율, 실시간 소통
- 토큰 ~2.5배, 속도 ~2배. 현재 Research Preview (실험적)
- **적합:** 코드/빌드 작업 (eng-* 에이전트)

### AG Agent Manager (현재 우즈덤 구조)
- GUI 기반. GPT/Gemini 페르소나 실행
- 파일 브릿지(`to_antigravity.md`) 비동기 통신
- 멀티모델 혼용 가능, 검증된 수동 흐름
- **적합:** 리서치/분석/재무 (fin-*, res-* 에이전트)

### 결론: 역할 분리 사용
> CC Agent Teams = 실행 레이어 / AG Agent Manager = 도메인 레이어
