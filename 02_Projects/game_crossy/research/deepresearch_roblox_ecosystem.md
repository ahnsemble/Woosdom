## Brain 보정 메모 (리서치 읽기 전 반드시 확인)

1. **라이브옵스 부담** — "스킬 1~2개 추가로 충분"은 과대평가. 밸런스 패치 부담 현실적으로 존재. 스킬 풀 8~10개로 시작할 것.

3. **Godot→Luau 전환** — "포팅"이 아니라 사실상 재개발. Knit/ProfileStore 세팅만 2~3주. Godot 프로토는 미니멀(1분 런) 검증 후 즉시 전환.
4. **플랫폼 종속성** — 리서치 미언급. 알고리즘/DevEx 환율 일방 변경 리스크 감수 전제.

## 섹션 4: 1인 개발 성공/실패 패턴

### 성공 공식
- 코어 루프 극단적 단순화 + RNG/소셜 요소
- 숏폼 바이럴 소재 내장
- 초기 성공 후 퍼블리셔/스튜디오 파트너십

### 실패 덫
1. Feature Creep — 정통 MMORPG 야심 → D1 리텐션 붕괴
2. 라이브옵스 부재 — 주간 업데이트 멈추면 알고리즘 차단
3. Creator Rewards 메타 오판 — 10분 세션 최적화 못하면 수익 반토막

### AI 활용
- Studio 내장 AI + MCP로 에셋/UI 자동화 가능
- **핵심 아키텍처(RemoteEvents, 메모리 관리)는 AI 코드 신뢰 불가 → 수동 통제 필수**

## 섹션 5: "Obby × 로그라이크" 적합성

### 시장 틈새
- 정통 Obby 쇠퇴 (수익화 한계 + 리플레이성 부재)
- 로그라이크 요소 Roblox 내 검증됨 (X3, VS 변형들)
- **하이브리드 = 낮은 진입장벽 + 무한 리플레이 + 깊은 수익화**

### 코어 루프
1. 절차적 생성 함정 구간(런) 돌파
2. 체크포인트마다 3택 무작위 스킬 선택
3. 추락 시 스킬 소멸, 메타 재화(영혼석)는 보존
4. 로비에서 영구 강화 해금

### 수익화 모델
- 게임패스: 영혼석 2배, 스킬 리롤 영구 1회
- 개발자 상품: 즉시 부활(스킬 유지), 전설 확률 부스트
- UGC/치장: 최고 기록 달성 전용 오라/이펙트

## 섹션 6: 실전 진입 전략

### Godot→Luau 전환 핵심
- OOP/Node 트리 → Luau 절차적/Metatables
- Signal → GoodSignal/SignalNet 이식
- 프레임워크: Knit (클라-서버), ProfileStore (데이터 저장)

### Pre-Launch 체크리스트
1. 썸네일 A/B 테스트 (CTR 1%+ 필수)
2. 태그/SEO 최적화 (Roguelike, Obby, Parkour, RNG)
3. 베타 커뮤니티 10명+ → D1 리텐션 20%+ 확인 후 출시

### Post-Launch (0→1,000 DAU)
1. Week 1: Sponsored Experiences 10K~20K Robux → 데이터 수집
2. 숏폼 파상 공세 (15초 카타르시스 영상)
3. Moments API 연동
4. 주간 스킬 1~2개 업데이트 (금요일 오후)
