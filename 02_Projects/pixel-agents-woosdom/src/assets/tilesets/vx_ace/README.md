# Tileset Assets (Not Included)

This project uses RPG Maker VX Ace compatible tilesets.
These are not included in the repository due to licensing restrictions.

## Required Tilesets
- Office Floors (A2)
- Office Walls (A4)
- Office Floors & Walls (A5)
- Office Furniture (B-C-D-E) x2

You can use any RPG Maker VX Ace compatible office tileset.
Place your PNG files in this directory with matching filenames.
